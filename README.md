# Massive

Massive gives you a basic infrastructure to parallelize processing of large files and/or data using Resque, Redis and MongoDB.

## Installation

Add this line to your application's Gemfile:

    gem 'massive'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install massive
